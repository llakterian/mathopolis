# Mathopolis: Build & Brighten! 🏗️✨
*A Fun Educational Game for Kids (Ages 5-10)*

Transform math practice into an exciting city-building adventure! Solve math problems to earn building blocks and construct your very own colorful city in Mathopolis.

## 🚀 Quick Start

### ➡️ [Play Now on GitHub Pages](https://llakterian.github.io/mathopolis/)

### 💻 Local Setup (One Command!)
```bash
git clone https://github.com/llakterian/mathopolis.git && open mathopolis/index.html
```

## 🎮 Game Features

### 🧮 **Math Learning Made Fun**
- **Interactive Problem Solving**: Answer math questions to earn building blocks
- **Progressive Difficulty**: Problems adapt as players advance
- **Instant Feedback**: Visual and audio cues for correct answers
- **Educational Focus**: Perfect for ages 5-10 to practice basic arithmetic

### 🏙️ **City Building Adventure**
- **Creative Construction**: Use earned blocks to build houses, castles, and more
- **Building Templates**: Pre-designed structures to guide young builders
- **Drag & Drop Interface**: Intuitive building mechanics for kids
- **Progress Tracking**: Visual progress bar shows city completion

### 🎨 **Engaging Visual Experience**
- **Animated Gradient Background**: Beautiful, ever-changing color schemes
- **Professor Puzzlo**: Friendly animated character guide
- **Sound Effects**: Satisfying audio feedback for achievements
- **Confetti Celebrations**: Reward animations for milestones
- **Night Mode**: Toggle between day and night themes

### 🌈 **Special Features**
- **Rainbow Run Minigame**: Break from math with fun bonus activities
- **Block Inventory System**: Track and manage earned building materials
- **Unlock System**: Discover new characters and building options
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## 🎯 Educational Benefits

- **Math Skills**: Addition, subtraction, and basic arithmetic
- **Problem Solving**: Critical thinking through building challenges
- **Creativity**: Open-ended city design and construction

## 🎮 How to Play

1. **Start Playing**: Open the game in your web browser
2. **Solve Math Problems**: Answer the equation shown on screen
3. **Earn Blocks**: Correct answers reward you with building blocks
4. **Build Your City**: Use blocks to construct buildings on the canvas
5. **Try Templates**: Use pre-made building designs or create your own
6. **Play Minigames**: Take breaks with the Rainbow Run bonus game
7. **Track Progress**: Watch your city grow with the progress bar
